# Design Tokens Reference

Complete list of all CSS custom properties available.

## Colors

### Background Colors

| Token | Default | Description |
|-------|---------|-------------|
| `--p-color-bg` | `#f6f6f7` | Page background |
| `--p-color-bg-surface` | `#ffffff` | Card/panel surface |
| `--p-color-bg-surface-hover` | `#f1f2f3` | Surface hover state |
| `--p-color-bg-fill-brand` | `#008060` | Primary buttons |

### Text Colors

| Token | Default | Description |
|-------|---------|-------------|
| `--p-color-text` | `#202223` | Primary text |
| `--p-color-text-secondary` | `#6d7175` | Secondary text |
| `--p-color-text-brand` | `#008060` | Brand text |

### Border Colors

| Token | Default | Description |
|-------|---------|-------------|
| `--p-color-border` | `#c9cccf` | Default border |
| `--p-color-border-brand` | `#008060` | Brand border |

## Spacing

| `--p-space-300` | `12px` |
| `--p-space-400` | `16px` |
| `--p-space-600` | `24px` |
